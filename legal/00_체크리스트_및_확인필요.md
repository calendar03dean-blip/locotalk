## A. 코드 확인으로 확정된 사실 (반영 완료)
| 항목 | 확정 내용 |
|---|---|
| 본인확인 연동 | **PortOne V2 + NHN한국사이버결제(KCP) 채널** (api.portone.io 결과조회). 사내문서상 "다날" 언급 존재 → 채널 최종확정 필요 |
| 본인확인 시행 | **현재 미시행**(`VERIFY_LIVE=false`), 출시 전 활성화 예정 |
| SMS 발송 | **솔라피(Solapi)** |
| 이메일(OTP) | **Amazon SES (서울 ap-northeast-2)** — 가비아 DNS amazonses 레코드와 일치 |
| 서버·DB | **Railway 호스팅 + PostgreSQL**, 리전 **US West(미국)** → 국외이전(미국) |
| 푸시 | **Expo Push + APNs(Apple)/FCM(Google)** |
| 정밀좌표 저장 | **디스크 미저장**(매칭 연산용 메모리만) → 위치약관 "지체없이 파기"와 부합 |
| 분석·광고 SDK | **없음** |
| 채팅 보관 | 무료 7일 / 프리미엄 90일 (PostgreSQL 차등보관) |
