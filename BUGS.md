# Locotalk 버그 트래킹

> **사용법**
> - 🔴 클로드B: 버그 발견 시 아래 양식으로 추가 후 `git push`
> - 🔵 클로드A: 수정 후 상태 변경 + commit hash 기록 후 `git push`
> - 🔴 클로드B: `git pull` 후 재테스트 → 결과 업데이트

---

## 버그 상태 표시
- `🔴 미수정` — B가 발견, A 수정 대기
- `🔵 수정중` — A가 작업 중
- `🟢 수정완료` — A가 수정, B 재테스트 요청
- `✅ 확인됨` — B가 재테스트 완료, 정상 동작
- `❌ 재현안됨` — B가 재테스트했으나 재현 불가
